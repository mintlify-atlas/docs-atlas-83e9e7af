---
title: 'Send due date reminder'
description: 'Send a due date reminder email to a library member listing their checked-out books.'
---

<Note>
  This endpoint does **not** require an authentication token. It is mounted before the token-verification middleware in `index.js` under the `/api` prefix.
</Note>

`POST /api/getDueDate`

Sends an HTML reminder email to the specified address using Nodemailer and Mailgen. The email lists the member's currently checked-out books with a reminder to return them by 12:00 PM to avoid late fees. A charge of 10 Rs per day applies for overdue returns.

<Warning>
  This endpoint sends a real email. Ensure `EMAIL` and `PASSWORD` are correctly configured in `backend/.env` before calling this endpoint. See [environment variables](/configuration/environment-variables) for setup instructions.
</Warning>

## Body

<ParamField body="userEmail" type="string" required>
  The email address of the library member to send the reminder to.
</ParamField>

## Response

<ResponseField name="msg" type="string">
  Confirmation message: `"you should receive an email"`.
</ResponseField>

### Error responses

| Status | Description |
| ------ | ----------- |
| `201`  | Email sent successfully. |
| `500`  | Failed to send email — check Nodemailer configuration. |

<CodeGroup>

```bash cURL
curl --request POST \
  --url http://localhost:5000/api/getDueDate \
  --header 'Content-Type: application/json' \
  --data '{
  "userEmail": "member@example.com"
}'
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/getDueDate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    userEmail: 'member@example.com',
  }),
});

const data = await response.json();
```

</CodeGroup>

```json 201 — success
{
  "msg": "you should receive an email"
}
```

