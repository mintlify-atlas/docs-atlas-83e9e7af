---
title: 'Get due date'
description: 'Retrieve the return due date for a checked-out book by ISBN.'
---

<Note>
  This is a utility endpoint. It does **not** require an authentication token
  because it is mounted before the token-verification middleware in `index.js`.
</Note>

`POST /api/getDueDate`

Returns the return due date for a specific checked-out book. This is used internally to populate due-date reminder emails and can also be called directly to query when a borrowed copy must be returned.

## Body

<ParamField body="isbn" type="string" required>
  ISBN of the book whose due date you want to retrieve.
</ParamField>

<ParamField body="userId" type="string">
  ObjectId of the user who borrowed the book. Providing this field scopes the
  lookup to a specific user when multiple borrowers hold the same ISBN.
</ParamField>

## Response

<ResponseField name="returnDate" type="string">
  The return due date in `YYYY-MM-DD` format.
</ResponseField>

<ResponseField name="isbn" type="string">
  ISBN of the queried book, echoed back in the response.
</ResponseField>

<ResponseField name="returned" type="boolean">
  Whether the item has already been returned.
</ResponseField>

### Error responses

| Status | Description |
| ------ | ----------- |
| `400`  | Required fields are missing or malformed. |
| `404`  | No active checkout found for the given ISBN (and user). |
| `500`  | Unexpected server error. |

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.libraease.com/api/getDueDate \
  --header 'Content-Type: application/json' \
  --data '{
  "isbn": "978-0-06-112008-4",
  "userId": "663f0a1b2c3d4e5f6a7b8c9d"
}'
```

```javascript JavaScript
const response = await fetch('https://api.libraease.com/api/getDueDate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    isbn: '978-0-06-112008-4',
    userId: '663f0a1b2c3d4e5f6a7b8c9d',
  }),
});

const data = await response.json();
```

</CodeGroup>

```json 200 — success
{
  "isbn": "978-0-06-112008-4",
  "returnDate": "2026-04-03",
  "returned": false
}
```

```json 404 — not found
{
  "message": "No active checkout found for the provided ISBN."
}
```
