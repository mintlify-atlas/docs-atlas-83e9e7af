---
title: 'Update book'
description: 'Update one or more fields of an existing book, looked up by ISBN.'
---

Finds a book by its ISBN and applies a partial update using the provided fields. Only the fields included in the `book` object are changed — all other fields remain unchanged.

<Warning>
  The book is located by ISBN (`id`). Providing an incorrect or non-existent ISBN will result in no document being updated and a `null` response in the `message` field.
</Warning>

## Authentication

<Note>
  This endpoint requires a valid JWT token passed in the request header as `token: <jwt-token>`. If the token is missing or invalid, the request will be rejected before reaching the controller.
</Note>

<ParamField header="token" type="string" required>
  JWT token issued at login.
</ParamField>

## Request

<ParamField body="id" type="string" required>
  ISBN of the book to update.
</ParamField>

<ParamField body="book" type="object" required>
  Object containing the fields to update. Only the provided fields are changed; all other fields are left as-is.

  <Expandable title="Updatable fields">
    <ParamField body="title" type="string">
      New title for the book.
    </ParamField>
    <ParamField body="author" type="string">
      Updated author name.
    </ParamField>
    <ParamField body="publisher" type="string">
      Updated publisher name.
    </ParamField>
    <ParamField body="year" type="number">
      Updated publication year.
    </ParamField>
    <ParamField body="genre" type="string">
      Updated genre or category.
    </ParamField>
    <ParamField body="quantity" type="number">
      Updated number of available copies.
    </ParamField>
    <ParamField body="images" type="string[]">
      Updated array of image URLs or base64-encoded strings.
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="message" type="object">
  The full updated book document as stored in the database after the update is applied.

  <Expandable title="Updated book object properties">
    <ResponseField name="isbn" type="string">
      International Standard Book Number.
    </ResponseField>
    <ResponseField name="title" type="string">
      Title of the book.
    </ResponseField>
    <ResponseField name="author" type="string">
      Name of the book's author.
    </ResponseField>
    <ResponseField name="publisher" type="string">
      Name of the publisher.
    </ResponseField>
    <ResponseField name="year" type="number">
      Year the book was published.
    </ResponseField>
    <ResponseField name="genre" type="string">
      Genre or category of the book.
    </ResponseField>
    <ResponseField name="quantity" type="number">
      Number of copies available.
    </ResponseField>
    <ResponseField name="images" type="string[]">
      Array of image URLs or base64-encoded strings.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="okk" type="boolean">
  `true` if the operation succeeded, `false` if it failed.
</ResponseField>

## Code examples

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.libraease.com/book/updatebook \
  --header 'token: <jwt-token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "id": "978-3-16-148410-0",
    "book": {
      "quantity": 8,
      "genre": "Classic Fiction"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.libraease.com/book/updatebook', {
  method: 'POST',
  headers: {
    'token': '<jwt-token>',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    id: '978-3-16-148410-0',
    book: {
      quantity: 8,
      genre: 'Classic Fiction',
    },
  }),
});

const data = await response.json();
console.log(data);
```

</CodeGroup>
