---
title: 'Get all books'
description: 'Retrieve every book in the library catalog, regardless of which librarian added it.'
---

Returns all books in the library system across all librarians. This is the primary endpoint for members to browse the full catalog.

<Note>
  To retrieve only the books added by the currently authenticated librarian, use the [Get books](/api/books/get-books) endpoint instead.
</Note>

## Authentication

<Note>
  This endpoint requires a valid JWT token passed in the request header as `token: <jwt-token>`. If the token is missing or invalid, the request will be rejected before reaching the controller.
</Note>

<ParamField header="token" type="string" required>
  JWT token issued at login.
</ParamField>

## Request

No request body is required.

## Response

<ResponseField name="data" type="object[]">
  Array of all book objects in the catalog. Internal fields (`_id`, `createdBy`, `__v`) are excluded from each object.

  <Expandable title="Book object properties">
    <ResponseField name="isbn" type="string">
      International Standard Book Number.
    </ResponseField>
    <ResponseField name="title" type="string">
      Title of the book.
    </ResponseField>
    <ResponseField name="author" type="string">
      Name of the book's author.
    </ResponseField>
    <ResponseField name="publisher" type="string">
      Name of the publisher.
    </ResponseField>
    <ResponseField name="year" type="number">
      Year the book was published.
    </ResponseField>
    <ResponseField name="genre" type="string">
      Genre or category of the book.
    </ResponseField>
    <ResponseField name="quantity" type="number">
      Number of copies available.
    </ResponseField>
    <ResponseField name="images" type="string[]">
      Array of image URLs or base64-encoded strings for the book.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="okk" type="boolean">
  `true` if the operation succeeded, `false` if it failed.
</ResponseField>

## Code examples

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.libraease.com/book/getallbooks \
  --header 'token: <jwt-token>'
```

```javascript JavaScript
const response = await fetch('https://api.libraease.com/book/getallbooks', {
  method: 'POST',
  headers: {
    'token': '<jwt-token>',
  },
});

const data = await response.json();
console.log(data);
```

</CodeGroup>
