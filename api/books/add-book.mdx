---
title: 'Add book'
description: 'Add a new book to the library catalog. The book is associated with the authenticated librarian.'
---

Adds a new book to the library catalog. The book is automatically linked to the authenticated user (librarian) via the `createdBy` field.

## Authentication

<Note>
  This endpoint requires a valid JWT token passed in the request header as `token: <jwt-token>`. If the token is missing or invalid, the request will be rejected before reaching the controller.
</Note>

<ParamField header="token" type="string" required>
  JWT token issued at login. Used to identify and authenticate the librarian.
</ParamField>

## Request

<ParamField body="isbn" type="string" required>
  International Standard Book Number. Must be unique across the catalog.
</ParamField>

<ParamField body="title" type="string" required>
  Title of the book.
</ParamField>

<ParamField body="author" type="string" required>
  Name of the book's author.
</ParamField>

<ParamField body="publisher" type="string" required>
  Name of the publishing house or publisher.
</ParamField>

<ParamField body="year" type="number" required>
  Year the book was published.
</ParamField>

<ParamField body="genre" type="string" required>
  Genre or category of the book (e.g., "Fiction", "Science").
</ParamField>

<ParamField body="quantity" type="number" required>
  Number of copies available in the library.
</ParamField>

<ParamField body="images" type="string[]" required>
  Array of image URLs or base64-encoded strings representing the book cover and any additional images.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Human-readable result message. `"Book added successfully"` on success.
</ResponseField>

<ResponseField name="okk" type="boolean">
  `true` if the operation succeeded, `false` if it failed.
</ResponseField>

## Code examples

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.libraease.com/book/addbook \
  --header 'token: <jwt-token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "isbn": "978-3-16-148410-0",
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "publisher": "Scribner",
    "year": 1925,
    "genre": "Fiction",
    "quantity": 5,
    "images": ["https://example.com/covers/great-gatsby.jpg"]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.libraease.com/book/addbook', {
  method: 'POST',
  headers: {
    'token': '<jwt-token>',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    isbn: '978-3-16-148410-0',
    title: 'The Great Gatsby',
    author: 'F. Scott Fitzgerald',
    publisher: 'Scribner',
    year: 1925,
    genre: 'Fiction',
    quantity: 5,
    images: ['https://example.com/covers/great-gatsby.jpg'],
  }),
});

const data = await response.json();
console.log(data);
```

</CodeGroup>
